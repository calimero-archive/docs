title: Calimero Endpoints
sidebar_position: 1

To interact with the Calimero private shard, we provide two endpoints:

- Calimero RPC Service Endpoint
- Calimero GraphQL Indexer Service Endpoint

Both endpoints can be found on the console dashboard:

![Workflow Diagram](https://github.com/calimero-is-near/docs/assets/39309699/42d758d9-089b-400f-9ba0-b85248118f4f)

You can copy these endpoints and integrate them into your application using the NEAR SDKs or the NEAR CLI. To interact with the shard, you need to generate an authorization token through the Calimero Dashboard or the NEAR Wallet Integration.
